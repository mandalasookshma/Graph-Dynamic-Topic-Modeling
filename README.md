Structured Temporal Topic Modeling with Graph Transformers and Adaptive Topic Weighting (Submitted to a Journal)
--
This repository contains the implementation of a dynamic topic modeling framework that integrates **variational inference**, **graph-based temporal modeling**, and a **CRF-inspired, HDP-based topic weighting mechanism**.

The model captures the evolution of latent topics over time while adaptively controlling topic usage across temporal slices.

---

## 🚀 Setup Instructions

### 1. Clone the repository
```bash
git clone https://github.com/mandalasookshma/Graph-Based-Dynamic-Topic-Modeling.git
cd Graph-Based-Dynamic-Topic-Modeling
```

### 2. Install dependencies
```bash
pip install -r requirements.txt
```

### 3. Make Script Executable
```bash
chmod +x run_dataset.sh
./run_dataset.sh
```

All outputs are stored in the `Results/` directory.
